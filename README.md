# @kandy-io/cpaas-sdk

### Npm Install

#### Package name :

`$ npm install @kandy-io/cpaas-sdk`

#### Github url :

Install with the beta branch or a particular build with the Github URL

`$ npm install https://github.com/Kandy-IO/kandy-cpaas-js-sdk#<build number or "beta"> --save`

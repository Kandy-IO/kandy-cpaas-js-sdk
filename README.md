# @kandy-io/cpaas-sdk

## Install

### Using npm :

`$ npm install @kandy-io/cpaas-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-cpaas-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/cpaas-sdk@4.5.0/dist/kandy.js"
  integrity="sha384-iw4lCQU4i9RRJC7L4iZ/DC45jnwnz8Igat9vERPFqTin/Y+3kMmwOjzIgPml0/FT"
  crossorigin="anonymous">
</script>
```

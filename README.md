# @kandy-io/cpaas-sdk

## Install

### Using npm :

`$ npm install @kandy-io/cpaas-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-cpaas-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/cpaas-sdk@4.7.0/dist/kandy.js"
  integrity="sha384-F0KGvCG60V/pr3j0LMTMvvx++PL7FlV0xf664Pt1ax2EYNAvol3w2Xy6FbEvZbbo"
  crossorigin="anonymous">
</script>
```

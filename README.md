# Kandy CPaaS JavaScript SDK

## Install

### Using npm :

`$ npm install @kandy-io/cpaas-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-cpaas-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/cpaas-sdk@4.33.0/dist/kandy.js"
  integrity="sha384-BxPeTJa2qnD09zC+4lPQ0t6NIR8yNJtJx5rVP+5MvlCgiyjCqESqRcze2l/ahK9e"
  crossorigin="anonymous">
</script>
```
